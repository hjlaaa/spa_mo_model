# CRC make_unique Integrated Pipeline Report

## 1. Scope

This task integrated CRC Stereo-CITE-seq RNA `var_names_make_unique()` handling
into a complete in-memory read/preprocess/model-forward pipeline.

The new flow is:

```text
read raw CRC h5ad
    -> in-memory RNA var_names_make_unique()
    -> align CRC_003 and CRC_006 shared RNA genes
    -> verify RNA/ADT obs_names and spatial coordinates
    -> construct data_dict with RNA + Protein
    -> reuse COSIE-style preprocessing
    -> get feature_dict / spatial_loc_dict / processed_data_dict
    -> initialize StageMultiModalModel
    -> initialize OT prior
    -> run one dry-run forward
    -> save lightweight summary files only
```

No formal processed h5ad files were saved. No original h5ad files were
overwritten. No model training was run. No full CRC UOT was constructed.

## 4. In-memory RNA var_names_make_unique

For each CRC RNA AnnData, the script executes the requested logic in memory:

```python
rna.var["gene_symbol_original"] = rna.var_names.astype(str)
rna.var["was_duplicate_gene_symbol"] = rna.var_names.duplicated(keep=False)
rna.var["gene_symbol_original_count"] = ...
rna.var_names = pd.Index(rna.var_names.astype(str))
rna.var_names_make_unique()
rna.var["gene_symbol_make_unique"] = rna.var_names.astype(str)
```

The explicit `pd.Index(...)` conversion is needed because the CRC backed h5ad
uses a categorical index for `var_names`; direct `var_names_make_unique()` on
that categorical index cannot add new suffix categories such as `RGS5-1`.

Important interpretation:

```text
MATR3-1, ABCF2-1, RGS5-1, etc. are engineering names generated by AnnData
var_names_make_unique(). They are not biological gene IDs.
```

The script does not:

- use `real_gene_name` to replace `var_names`;
- fabricate gene IDs;
- aggregate duplicated count columns;
- write any modified RNA h5ad.

## 15. Warnings / notes

During CRC preprocessing, Scanpy warned that some cells have zero counts in the
selected dry-run RNA subset. The dry-run still completed and produced finite
loss. This should be monitored when scaling beyond the first 1000 spots and
first 3000 shared genes.

The artificial suffix count in the full shared gene list is 19, but the first
3000 selected shared genes used by the dry run contained 0 artificial suffix
genes. This is a consequence of using the first 3000 shared genes for speed.
Formal experiments should revisit `max_shared_genes` and preferably use a
biologically motivated gene selection or the full shared gene set after
resource planning.
